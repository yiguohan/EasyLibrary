# EasyLibrary
## Download
Step1: Add it in your root build.gradle at the end of repositories:
```
allprojects {
    repositories {
        ...
        maven{ url 'https://jitpack.io'}
    }
}
```
Step2: Add the dependency
```
dependencies {
    compile 'com.github.yiguohan:EasyLibrary:beta0.1'
}
```
